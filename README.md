**Project Overview**
This repository contains two practical applications of Machine Learning:

**1. Breast Cancer Classification**
*   **Objective:** Predict whether a tumor is Malignant or Benign based on clinical features.
*   **Algorithm:** Support Vector Machine (SVM) with StandardScaler.
*   **Output:** Generates `performance.csv` and `cancer_predictions.csv`.

**2. Insurance Spend Regression**
*   **Objective:** Estimate average cancer-related insurance spending based on regional and demographic factors.
*   **Algorithm:** Random Forest Regressor with OneHotEncoding for categorical data.
*   **Output:** Generates `performance2.csv` and visualization of Actual vs. Predicted values.

**Tech Stack**
*   **Language:** Python (IDLE)
*   **Libraries:** Pandas, Scikit-Learn, Matplotlib, Seaborn
